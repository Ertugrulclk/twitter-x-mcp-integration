## 🎯 Neden Bu Projeyi Yaptım?

Bir **Vibe Coder** olarak amacım, projelerimi ve fikirlerimi **hızlı, etkili ve otomatik** bir şekilde dünyayla paylaşmak. Sosyal medya içerik yönetimi zaman alıcı bir süreç — ama ya bunu doğrudan kod editöründen, AI ile konuşarak yapabilseydiniz?

İşte tam olarak bunu yaptım.

## 🔗 Nasıl Çalışıyor?

**MCP (Model Context Protocol)** kullanarak VS Code içindeki GitHub Copilot'a Twitter/X platformuyla doğrudan iletişim kurma yeteneği kazandırdım.

```
Ben (Vibe Coder) → GitHub Copilot (AI) → MCP Server → Twitter/X API → Tweet Paylaşımı
```

### Süreç şöyle ilerliyor:

1. **VS Code'da çalışırken** bir fikir veya içerik paylaşmak istiyorum
2. **GitHub Copilot'a doğal dilde** söylüyorum: *"Şu konuda bir tweet at"*
3. **MCP Server** bu isteği alıp Twitter/X API'sine iletiyor
4. **İçerik anında yayınlanıyor** — editörden çıkmadan!

## 🚀 Vibe Coding Yaklaşımı

Bu proje, modern yazılım geliştirmenin ruhunu yansıtıyor:

- **AI ile doğal iletişim** — Kod yazmak yerine konuşarak yönlendirme
- **Hızlı prototipleme** — Fikirden üretime dakikalar içinde geçiş
- **Otomasyon odaklı** — Tekrarlayan görevleri AI'a devret

## 🛠️ Teknolojiler

| Teknoloji | Kullanım |
|-----------|----------|
| **MCP (Model Context Protocol)** | AI-araç iletişim katmanı |
| **GitHub Copilot** | AI asistan |
| **VS Code** | Geliştirme ortamı |
| **Twitter/X API** | Sosyal medya entegrasyonu |
| **Node.js** | MCP Server runtime |
